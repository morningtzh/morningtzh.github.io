Git分支使用方法

分支概要

分支使用tip:

  1. 设三个分支, master/release/develop
  2. master和release 设置为只能管理员merge
  3. 可以把develop设置为默认分支
  4. 未来在网页端merge选上 Squash commits when merge request is accepted.
  5. merge到master分支上需要打上tag
  6. feature分支大家可自由创建~ 命名为 feature/xxx
  7. feature分支合并至develop时, 和以上一样选择. 最后次合入时同时选上 Delete source branch when merge request is accepted. 以删除feature分支
